# Switch

## Definition

It refers to an ideal switch that connect two buses.

```{image}  /_static/Switch.svg
:alt: Switch diagram
:width: 300px
:align: center
```

## Equations

The associated equations are:
```{math}
\left\{
    \begin{aligned}
        \underline{I_1} &= - \underline{I_2}\\
        \underline{V_1} &= \underline{V_2}\\
    \end{aligned}
\right.
```
